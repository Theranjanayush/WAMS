# Web-Based Automated Manufacturing System (WAMS)

## 📌 Overview
This project is an implementation of a **Software Requirements Specification (SRS)** developed during my Software Engineering course.

The application simulates a real-world **manufacturing management system** used by companies that produce goods based on dealer orders and manage raw materials from suppliers.

## 🧩 Problem Statement

This application is designed for manufacturing companies that:
- Produce goods based on dealer orders  
- Manage raw materials from suppliers  
- Maintain stock and transaction records  

The system handles:
- Billing management  
- Dealer requests  
- Parts inventory  
- Product supply and stock  
- Product transactions  
- Supplier quotations  
- Supplier details  

---

## ⚙️ System Workflow

1. Dealers place product orders  
2. System checks product availability  
3. If stock is insufficient:
   - Raw materials are requested from suppliers  
   - Suppliers provide quotations  
4. Company reviews and selects quotations  
5. Manufacturing process is initiated  
6. Product stock is updated  
7. Dealers are notified for order fulfillment  
8. Billing and transactions are recorded  

Additionally:
- The system uses past sales data to assist in **stock planning**
- Includes a basic **decision-support mechanism** for management
